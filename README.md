# Diabetes Prediction Using KNN

## Overview
This project uses the K-Nearest Neighbors (KNN) algorithm to predict whether a patient is Diabetic or Non-Diabetic based on medical measurements.

## Dataset
Pima Indians Diabetes Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model
- Algorithm: K-Nearest Neighbors (KNN)
- Best K Value: 18
- Accuracy: 75.32%

## Conclusion
The KNN model was successfully trained and evaluated for diabetes prediction.
